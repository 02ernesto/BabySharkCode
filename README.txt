Baby Shark Code

Video juego llamado Baby Shark Code (pequeño homenaje al editor de código que utilizamos “Visual Studio Code”) El cuál es el predador y tiene que comer a los lenguajes de programación con poco futuro. 

GUÍA PARA JUGAR
Se pueden utilizar las flechas de desplazamiento < - izquierda, derecha, -> arriba ^,  abajo¥ . Aunque también se tiene la opción de utilizar el Mouse.  


OBJETIVO I
El objetivo es comer a los pequeños logotipos en forma de peces, de los lenguajes de programación contrarios a lo que vemos en IronHack como PHP, PERL,  DEPHI. Basta con tocarlos para comerlos. 

Se pueden perder vidas al ser tocados por lenguajes de programación más robustos, que representan competencia real a JavaScript. Como Java y Phyton. 

OBJETIVO II 
Mantener entretenido a mi sobrino el mayor tiempo posible. 


Al juntar 80 puntos, sale el primer jefe. 
Al juntar 250 puntos, sale un aliado, llamado GO. El nuevo lenguaje de programación que viene pisando fuerte. 
Se gana al llegar a los 400 puntos.
